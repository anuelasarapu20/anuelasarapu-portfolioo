Project 2

Project Description

This project develops a secure online e-voting system that uses biometric authentication (face and iris recognition) to verify voters. Users register by scanning their face and iris, then log in with a username and password. During login, their biometric data is matched with the stored information to allow voting. To prevent multiple votes, the system checks the database and ensures that each user votes only once. This system enhances the security, accessibility, and integrity of the voting process.

Project Goals

The goal of this project is to develop a secure online e-voting system that uses biometric authentication (face and iris recognition) to verify voters. The system ensures that only registered users can log in and cast their vote after their identity is successfully authenticated. It also prevents multiple voting by checking the database to confirm that each user can only vote once. Ultimately, the project aims to enhance the security, accessibility, and integrity of the voting process, making elections more reliable and transparent.

Challenges

The challenges in this project include ensuring accurate face and iris recognition under varying lighting conditions and preventing biometric spoofing. Additionally, it is crucial to maintain high security to protect user data from breaches, prevent unauthorized access, and guarantee that only registered users can vote. The system must also be scalable to handle large volumes of real-time votes while maintaining performance. Managing user privacy during biometric data collection and ensuring the system remains operational without failures during elections are other critical challenges to address.
